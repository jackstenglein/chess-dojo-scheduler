“Everything in the cohort worked well together to help me better my chess. Polgar mates helped me with piece awareness and board awareness, 1001 helped tactics familiarity and broadening and calculation, and the Most Instructional games really brought it altogether for me. The program is amazing, I didn't think I could improve really being that I'm an old fart but it's happening.”

- Spiderzlatan on graduating to 1200-1300

Tablebases - Jack continues to build out the Dojo database. His latest improvement is that we now have the ability to solve all seven-piece (or less) endgames. It’s an incredibly powerful tool that allows for easy navigation of complex endgames. We now have a database that is just as good as Chessbase – at a fraction of the cost! And soon we will have an opening move trainer which will allow dojoers to create an opening file and then instantly train it. Previously one had to create a file in Chessbase and then import that file into Chessable (a process which Chessable deliberately made difficult as it wants you to buy their courses). The Dojo will make it so much easier and all in one place.
