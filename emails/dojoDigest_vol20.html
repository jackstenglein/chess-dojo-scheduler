Server Roles - Dojoers used to have to manually update their cohort roles on discord when they graduated. Bulk Download - We now have an option to bulk download our games and directories. On your profile, select the games/directories you want to download, then in the editor window that pops up click the download icon (seem image below). A dialog will pop up which allows you to customize what is included in the download (second image).


Save Variation as a Comment - There has been a long-standing issue in the Dojo game viewer that if you add a new variation to someone else's game and leave a comment, then the comment would be lost when the owner opened the game again. This issue is now fixed! When you add a new variation to someone else's game, a warning symbol will appear. Hovering over the warning will show that the variation is unsaved. For example, in the first screenshot, I have added 1.d4 as a variation, which is unsaved. Right clicking on any move in the unsaved variation will allow you to save it using the save variation as comment button (second screenshot). Once the variation is saved, you will see it in the PGN text notated with your profile picture. Hovering over the picture will show who suggested the variation (third screenshot). Your variation will also appear in the comments tab, and you can navigate through the variation by clicking the moves in the comment tab just like you can in the PGN text (fourth screenshot). If you add a regular text comment to an unsaved variation, the variation will be automatically saved as well. Finally, if you don't want to see suggested variations from other users in the PGN text, you can disable that using the new Display other users' suggested variations in PGN text option in the settings tab (fifth screenshot). If you uncheck this option, your own suggestions will still appear in the PGN text, but other users' will only be visible in the comments tab.
